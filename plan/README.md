# PredixScore Plan

이 폴더는 실제 개발 진행 상태를 추적하는 작업판이다.

작업 규칙:

- 작업을 시작하기 전에 `plan/progress.md`를 확인한다.
- 작업 중 범위가 바뀌면 `plan/progress.md`를 갱신한다.
- 작업이 끝나면 완료한 항목, 검증 결과, 다음 작업을 기록한다.
- 제품 방향은 `docs/product-plan.md`를 기준으로 한다.
- 개발 순서는 `docs/implementation-plan.md`를 기준으로 한다.

현재 기본 트랙:

- `Track A`: 블록체인 없이 Supabase paper trading 먼저
